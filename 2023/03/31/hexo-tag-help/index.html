Tag Plugins

Label 标签

使用方法

  • [class]: 可选参数. 支持的参数: default | primary | success | info | warning | danger.
  • 如果没有指定，将使用浏览器的默认样式，这在不同的浏览器中可能不同.
  • text : @text 可以指定有空格还是没有空格.
  • 例如: success @text 和 success@text 是等价的

Tabs 标签页

NexT config配置

使用方法

  • [Unique name]：标签块标签的唯一名称，不带逗号。
    • 将在#id’s中用作每个标签及其索引号的前缀。
    • 如果名称中有空格，对于生成#id，所有空格将被破折号取代。
    • 仅对于当前的post/page URL，必须是唯一的！
  • [index]：活动选项卡的索引号。
    • 如果没有指定，第一个选项卡(1)将被选中。
    • 如果index为-1，则不选择任何选项卡。这有点像剧透。
    • 可选参数。
  • [Tab caption]：当前标签的标题。
    • 如果没有指定标题，带有标签索引后缀的唯一名称将被用作标签标题。
    • 如果没有指定标题，但指定了图标，标题将为空。
    • 可选参数。
  • [@icon]：Font Awesome图标名称。
    • 可以指定有或没有空格;如 ‘Tab caption @icon’ 与 ‘Tab caption@icon’等同。
    • 可选参数。
